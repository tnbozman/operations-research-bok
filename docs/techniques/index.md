# OR Techniques

This section provides in-depth coverage of the core Operations Research techniques used across engineering and military analysis. Each technique includes:

- A **lecture** module covering theory, mathematical formulation, and key concepts
- **Real-world practical examples** with complete, runnable Python code
- **Interactive exercises** for hands-on learning
- An **Agent Prompt** you can paste directly into an AI assistant to generate or extend the content

## Techniques Covered

| Technique | Description |
|-----------|-------------|
| [**Linear Programming**](linear-programming/index.md) | Optimizing a linear objective subject to linear constraints |
| [**Integer Programming**](integer-programming/index.md) | Optimization where some or all variables are restricted to integers |
| [**Network Analysis**](network-analysis/index.md) | Modeling flows, routing, and connectivity in networked systems |
| [**Simulation**](simulation/index.md) | Modeling stochastic systems to estimate performance and risk |
| [**Queueing Theory**](queueing-theory/index.md) | Analyzing waiting lines and service systems |
| [**Decision Analysis**](decision-analysis/index.md) | Structuring decisions under uncertainty using probability and utility theory |
| [**Game Theory**](game-theory/index.md) | Modeling strategic interactions between rational agents |
| [**Heuristics & Metaheuristics**](heuristics-metaheuristics/index.md) | Approximate methods for large or intractable optimization problems |
| [**Stochastic Processes**](stochastic-processes/index.md) | Modeling systems that evolve probabilistically over time |
